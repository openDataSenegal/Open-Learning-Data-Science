# Projet — Analyse de données transactionnelles

## Objectif

Ce projet a pour objectif d’évaluer la capacité à réaliser une **analyse complète de données transactionnelles** en utilisant un outil d’assistance par intelligence artificielle.

L’analyse devra être réalisée **exclusivement avec Claude Code**, sans écriture manuelle de code par l’utilisateur.

---

## Instructions

1. **Prenez connaissance du projet** en consultant attentivement le document `projet_senior_data_analyst.pdf`, qui décrit les objectifs de l’analyse ainsi que les différentes questions auxquelles répondre.

2. **Téléchargez l’archive contenant les données** à partir du lien suivant :
   **[https://drive.google.com/file/d/1pEro5doB8Ss6Te_PxX4Ao7-cS5ZId7My/view?usp=sharing]**

3. **Extrayez le contenu de l’archive téléchargée.**
   Après extraction, vous trouverez le fichier de données suivant :
   `transaction_data.csv`

4. **Réalisez l’ensemble des analyses demandées** dans le document `projet_senior_data_analyst.pdf` en utilisant **exclusivement Claude Code**.

5. **Produisez un rapport final au format PDF** présentant de manière structurée :

   * la méthodologie d’analyse utilisée,
   * les résultats obtenus,
   * les visualisations et graphiques pertinents,
   * les interprétations et conclusions tirées des analyses.

6. **Contraintes de réalisation :**

   * La durée totale de réalisation ne doit **pas excéder une heure**.
   * **Aucune ligne de code ne doit être écrite manuellement par l’utilisateur.**
   * L’ensemble du travail analytique doit être généré via **Claude Code**.

---

## Exemple de prompt

Voici un exemple de prompt pouvant être utilisé pour démarrer l’analyse :

> Tu es un **Senior Data Analyst**.
> Analyse le document `projet_senior_data_analyst.pdf`, qui décrit les analyses de données à réaliser sur les transactions contenues dans le fichier `transaction_data.csv`.
> Réalise l’ensemble des analyses demandées dans le document et produis un **rapport final structuré au format PDF**, incluant les graphiques, les résultats analytiques ainsi que les conclusions issues de l’étude.
