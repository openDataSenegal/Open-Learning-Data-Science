Python fournit des structures de donn√©es int√©gr√©es tr√®s puissantes et faciles √† utiliser. Voici les principales √† conna√Ætre pour toute analyse de donn√©es :

Une **liste** est une collection ordonn√©e et **modifiable** d‚Äô√©l√©ments. Elle peut contenir des types de donn√©es vari√©s (entiers, cha√Ænes, listes, etc.).
